# 多重ゲーム構造の統合シミュレーション

博士論文「いじめ問題の多重ゲーム構造」付録 A・B のシミュレーション実装と、
付録 B の全実験（E1–E4）を再現するためのコードです。

## 動作環境
- Python 3（標準ライブラリのみ。外部パッケージ不要）
- 乱数シードは試行番号で管理しており、結果は完全に再現可能です。

## ファイル構成
| ファイル | 内容 |
|---|---|
| `multilayer_sim_v2.py` | コアモデル（付録 A の仕様・付録 B.9 掲載分）。主体・評価関数・結合規則 C1–C10・介入レバー L1–L4 を実装。 |
| `reproduce_appendix_B.py` | 付録 B の実験ハーネス（E1–E4）。コアモデルを読み込んで各図表を再生成する。 |

## 実行方法
```bash
# コアモデル単体のデモ（基準条件などの一覧）
python3 multilayer_sim_v2.py

# 付録Bの各実験を再現
python3 reproduce_appendix_B.py e1    # B.3   基準条件における固定化
python3 reproduce_appendix_B.py e2    # B.4   16構成の必要条件構造
python3 reproduce_appendix_B.py e3    # B.5   介入タイミングと遅延の費用
python3 reproduce_appendix_B.py e41   # B.6.1 判断余力供給の臨界性
python3 reproduce_appendix_B.py e42   # B.6.2 一次元±50%感度
python3 reproduce_appendix_B.py e43   # B.6.3 大域感度（LHS）
python3 reproduce_appendix_B.py e44   # B.6.4 選択規則の頑健性（ロジット）
python3 reproduce_appendix_B.py all   # すべて連続実行
```

## 各実験と論文の対応
- **E1（B.3）**: 無介入下で固定化が支配的帰結（約 89.5%）となり、ほぼ二重ロックインで終局。
- **E2（B.4）**: 早期に固定化率を 5% 未満に抑える構成（L2 を含む 6 構成）と、
  事後に解除可能な構成（L1L2L3・L1L2L3L4 の 2 構成のみ）を特定。
- **E3（B.5）**: 四レバー同時供給。観察期間は「介入開始から 60 ターン」。
  遅延に伴い被害曝露総量が増大（t=0:7／t=50:146／t=80:203）。
  解決ターン（最終停止ターン、A.9 の定義）は早期 約 7、ロックイン形成後 約 36–38。
- **E4.1（B.6.1）**: L2 供給強度の臨界性。強度 0.8（87.0%）と 0.9（15.7%）の間で急峻に転移。
- **E4.2（B.6.2）**: 主要 14 パラメータの一次元±50% 感度。大半で主要主張が成立し、
  境界条件（案件負荷過大、対応・組織化コスト小 等）でのみ崩れる。
- **E4.3（B.6.3）**: ラテン超方格法による大域感度。主要主張の成立率を報告。
  ※モンテカルロ推定のため、`samples`・`n` を大きくするほど安定します（論文値は samples=150, n=300）。
- **E4.4（B.6.4）**: ロジット選択（β=10/20/40）下でも必要条件構造が保持されることを確認。
